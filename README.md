# Jira Transparency Dashboard

A mobile-first, React.js + Node.js transparency dashboard for Jira exports.

## Structure

- `backend/`: Node.js API server with file upload and metric engine.
- `frontend/`: React.js dashboard UI.

## Getting Started

### Backend

```bash
cd backend
npm install
npm start
```

### Frontend

```bash
cd frontend
npm install
npm start
```

## Notes

This scaffold includes upload handling, validation, and a starter dashboard layout for Jira export data.
